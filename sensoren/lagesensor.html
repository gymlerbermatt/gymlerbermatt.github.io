Lagesensor ​

Beispiel: synchrone Ausrichtung ​

Der Gyrosensor kann - ähnlich wie ein Kompass - die Orientierung und somit eine Drehung um eine gegebene Achse messen. Diese Achse ist auf dem Sensor durch einen roten Punkt dargestellt. Damit der Sensor nicht umkippt, steht er in der Abbildung auf einem breiten Fuss.
Im folgenden Beispiel soll der am Motor befestigte Zeiger immer gleich ausgerichtet sein wie der Sensor (im Bild nach rechts).

Der Gyrosensor kann mit der Funktion speed() übrigens auch Beschleunigungen messen und so zum Beispiel feststellen, ob der Roboter gerade über die Tischkante fährt und abstürzt.

python
#!/usr/bin/env pybricks-micropython
from pybricks import ev3brick as brick
from pybricks.hubs import EV3Brick
from pybricks.ev3devices import (Motor, TouchSensor, ColorSensor, InfraredSensor, UltrasonicSensor, GyroSensor)
from pybricks.parameters import Port, Stop, Direction, Button, Color
from pybricks.tools import wait, StopWatch, DataLog
from pybricks.robotics import DriveBase
from pybricks.media.ev3dev import SoundFile, ImageFile

gyro_sensor = GyroSensor(Port.S1)
motor = Motor(Port.A)

while True:
    winkel = gyro_sensor.angle()   # Lage (Winkel) des Sensors einlesen
    print(winkel)                  # Ausgabe zum Überprüfen des Winkels
    motor.track_target(winkel)     # Motor zum entspr. Winkel bewegen
    wait(100)                      # 1/10 Sekunde warten
